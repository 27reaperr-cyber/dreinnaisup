# 🤖 Dreinn VPN — Telegram Business Bot

AI-поддержка для клиентов Dreinn VPN через Telegram Business.

---

## ⚡ Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка `.env`
```bash
cp .env.example .env
```
Заполните файл `.env`:

| Переменная      | Описание                                        |
|-----------------|-------------------------------------------------|
| `BOT_TOKEN`     | Токен от [@BotFather](https://t.me/BotFather)   |
| `ADMIN_ID`      | Ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)) |
| `ONLYSQ_API_KEY`| API ключ OnlySq (если есть, иначе оставить пустым) |

### 3. Запуск
```bash
npm start
# или с авто-перезапуском (Node 18+)
npm run dev
```

---

## 🔗 Привязка к Telegram Business

1. Откройте **Telegram → Настройки → Telegram Business → Чат-бот**
2. Введите username вашего бота
3. Включите **«Управлять сообщениями»**
4. Бот пришлёт уведомление об успешном подключении

## 🛠 Команды админа

Доступны только из личного чата с ботом:

| Команда | Действие |
|---------|----------|
| `/start` | Открыть панель управления |

В панели управления:
- **Сменить модель** — выбрать модель из списка или ввести вручную
- **Изменить скилл** — задать системный промпт для ИИ
- **Список моделей** — получить актуальный список из OnlySq API
- **Статистика** — пользователи, сообщения, отключения ИИ
- **Пользователи** — список тех, кто отключил ИИ, с возможностью включить

---

## 💬 Команды пользователей

Работают в чате с вами (через Telegram Business):

| Команда | Действие |
|---------|----------|
| `/disable` | Отключить ИИ и связаться с оператором |
| `/enable` | Снова включить ИИ-помощника |

---

## 📊 Схема работы

```
Пользователь пишет
       ↓
   ИИ отключён? → ДА → Молчим (оператор отвечает вручную)
       ↓ НЕТ
  Запрос в OnlySq AI
       ↓
  ИИ отвечает пользователю
  (с подсказкой /disable внизу)
       ↓
  Пользователь недоволен → /disable
       ↓
  Уведомление админу → ручной ответ
```

## 🗃 База данных

SQLite файл `dreinn.db` создаётся автоматически.

| Таблица                | Назначение                         |
|------------------------|------------------------------------|
| `settings`             | Модель и системный промпт          |
| `conversations`        | История диалогов с пользователями  |
| `disabled_users`       | Пользователи с отключённым ИИ      |
| `business_connections` | История Business подключений       |
